\section*{Abstract}
Temporal-causal reasoning over multi-session dialogues is a critical capability
for memory-augmented LLM agents. However, existing memory systems suffer from
two coupled weaknesses: they conflate session time with event occurrence time,
and they cannot propagate temporal inferences across retrieval rounds to locate
indirectly addressable evidence. We introduce \sys{}, an RL-trained memory
evidence synthesis framework, which reformulates temporal memory search as
iterative evidence synthesis. The core mechanism is a Temporal Evidence Pool
that accumulates structured event records with separate event-time and
session-time fields. At ingestion, \sys{} applies lightweight deterministic
normalization to resolve relative temporal expressions; at search time, a
Planner decomposes the query into retrieval cues, and a trainable Synthesizer
extracts temporally grounded events from retrieved sessions and judges whether
the pool is sufficient to answer the query. If not, the inferred events feed
back to the Planner to drive the next retrieval round. Since the Synthesizer's
extraction and sufficiency decisions cannot be reliably elicited through
prompting alone, we train it with GRPO using multi-level rewards covering output
format, evidence compactness, and answer correctness. In particular, a
per-round verdict reward supervises the sufficiency decision at each retrieval
step, providing dense process supervision for the stop-or-continue choice. We
evaluate \sys{} on LoCoMo, LongMemEval, and \bench{}, a new benchmark for
temporal-causal reasoning in conversational memory. \sys{} achieves
state-of-the-art performance across all three benchmarks, improving over the
strongest prior memory methods by 12.5, 11.8, and 31.0 percentage points on
LoCoMo, LongMemEval, and \bench{}, respectively, while using 74.0\% fewer
evidence tokens than Mem0 on average over LoCoMo and LongMemEval.
