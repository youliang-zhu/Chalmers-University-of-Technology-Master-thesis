# Agents

This directory contains context and planning documents for agent-assisted thesis
writing.

- `project_contexts.md`: primary project context for migrating the NanoMem NeurIPS paper
  into the Chalmers master's thesis template.
- `resources/history_paper_commit.md`: distilled writing material recovered from
  the NanoMem paper commit history.

Keep final thesis prose in `paper/include/`. Keep planning notes, source
mapping, and agent instructions here.
